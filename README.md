# Energy Capacity Analysis/Power-Generation-Capacity (India)

This project analyzes India's power generation capacity dataset using Exploratory Data Analysis (EDA) and Machine Learning.

## Objective
To understand trends, patterns, and relationships between different energy sources and predict total installed capacity.

## Key Insights
- Thermal energy dominates the overall capacity
- Renewable energy is steadily increasing
- Total capacity shows consistent growth over time
- Strong correlation exists between energy sources

## Techniques Used
- Data Cleaning & Preprocessing
- Visualization (Seaborn, Matplotlib)
- Correlation Analysis
- Machine Learning (Linear Regression, Random Forest)

## Key Observation
Very high model accuracy revealed strong feature dependency and potential data leakage, highlighting the importance of feature selection.
